# Редактор и файлы описаний слухов

Основной способ изменения данных — кнопка **«Открыть редактор»** в настройках сканера слухов. Редактор предназначен и для обычного пользователя: значения категорий и тиров выбираются из списков, а данные проверяются до записи.

## Разделение данных

```text
rumor_catalog.default.json  встроенный каталог приложения
rumor_catalog.user.json     пользовательские изменения
rumor_catalog.user.json.bak предыдущая пользовательская версия
```

При загрузке пользовательские записи накладываются на встроенные по `id`. Поэтому обновление приложения может добавлять новые встроенные карты, не уничтожая пользовательские заметки.

Если рядом с приложением остался редактировавшийся в старой версии `rumor_catalog.json`, при первом запуске его изменённые и дополнительные записи автоматически переносятся в пользовательский файл. Новые встроенные записи при этом не отключаются.

## Возможности редактора

- поиск и фильтры по категории и тиру;
- добавление, отключение и повторное включение записей;
- несколько OCR-фраз на одну запись;
- категории: экспедиция, босс, уникальная карта;
- выразительные тиры: `S+`, `S`, `A`, `B`, `C`, `D`, без тира;
- отдельные русские и английские поля;
- карта, тип карты, модификаторы, награда, описание, заметка и rating notes;
- автоматическая проверка уникальности ID и нормализованных OCR-фраз;
- атомарное сохранение и резервная копия.

Отключённые записи остаются внизу списка. Для встроенной записи в пользовательский файл сохраняется только её ID в `disabledIds`; пользовательская запись сохраняется вместе со своими полями, поэтому её тоже можно включить обратно. Новые OCR-алиасы из обновлений приложения автоматически добавляются к существующим пользовательским правкам того же ID.

## Поля записи

```json
{
  "id": "expedition-endless-cliffs",
  "phrases": ["Endless cliffs", "Endless cllffs"],
  "kind": "expedition",
  "titleRu": "Бесконечные утёсы",
  "titleEn": "Endless cliffs",
  "mapRu": "Скалистый мыс",
  "mapEn": "Craggy Peninsula",
  "mapTypeRu": "Карта экспедиции",
  "mapTypeEn": "Expedition map",
  "modsRu": "Важные модификаторы",
  "modsEn": "Important modifiers",
  "resultRu": "Ожидаемая награда",
  "resultEn": "Expected reward",
  "detailRu": "Краткое описание",
  "detailEn": "Short description",
  "noteRu": "Заметка сообщества",
  "noteEn": "Community note",
  "ratingNotesRu": "Почему выставлен этот тир",
  "ratingNotesEn": "Why this tier was assigned",
  "rating": "A",
  "tags": ["expedition"],
  "priority": 70
}
```

## Валидация

Редактор не сохранит данные, если:

- ID пустой, содержит неподдерживаемые символы или повторяется;
- не указана ни одна OCR-фраза;
- нормализованная OCR-фраза уже принадлежит другой записи;
- фраза слишком короткая для безопасного сопоставления;
- выбрана неизвестная категория или тир;
- приоритет не является целым числом.

JSON всё ещё можно открыть вручную через вторичную кнопку для опытных пользователей. При повреждении файла приложение продолжает использовать последнюю корректную версию и пишет предупреждение в `logs\rumors.log`.
